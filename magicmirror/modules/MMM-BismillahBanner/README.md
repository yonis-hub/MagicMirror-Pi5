# MMM-BismillahBanner

Simple static top-bar banner to display:

`بِسْمِ ٱللهِ ٱلرَّحْمٰنِ ٱلرَّحِيْمِ`

## Example config

```js
{
  module: "MMM-BismillahBanner",
  position: "top_bar",
  config: {
    stylePreset: "classical-naskh",
    fontScale: 1.25,
    textColor: "#ffffff",
    showLigature: false,
    showTransliteration: false
  }
}
```

## Style presets

- `classical-naskh` (default)
- `thuluth`
- `compact`
